В папке Trying_to_understand..... лежат файлы с кодом проверки тестов пользователя
Разделены на две папки: for c++ (для Gtest на c++)
		       for python (для PyTest на питоне)


Prog -- программа, в которой вычисляется значение функции/абстрактного кода (задачи, одним словом)

C/Py_tests -- программа с тестами студентов (имитирует написанные пользователем для проверки программы тесты)

C/Py_check_tests -- программы парсеры кодов программ C/Py_tests, проверяющие корректность и достаточность написанных пользователем тестов
